You want to adopt a small kitteh irccat ?  Read this guide :)

#1 Install the gem

Just do: gem install irc_cat

#2 Configuration

Create config.yml. Edit it with your prefered editor:

  • irc:
      • host: is where you irc cat will be connected (host). For exemple, irc.tty2.org or irc.freenode.net
      port: is the port of the IRC Server. Usually, '6667'
      nick: is the nickname of your bot.
      channel: is the channel where you bot will stay and send messages.
  • http:
      enabled: Enable or not the HTTP server. Values: true, false
      host: is where the http server will bind. Use 0.0.0.0 to bind on all interfaces.
      port: is the port of the http server.
      github: Enable or not the github support. Values: true, false
      send: Enable or not the basic http send support. Values: true, false
  • tcp:
      enabled: Enable or not the TCP server. Values: true, false
      host: is where the tcp server will bind. Use 0.0.0.0 to bind on all interfaces.
      port: is the port of the tcp server.

View the example configuration file.

#3 Run

Just run irccat -c yourconfig.yml !
